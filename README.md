# Alky Wallet

## Descripción breve
Alky Wallet es una aplicación web de simulación de banca personal creada con HTML, CSS y JavaScript. Permite navegar entre diferentes pantallas de login, transacciones, depósitos y envío de dinero usando una interfaz tipo Bootstrap.

## Principales funciones
- Autenticación básica de usuario en `login.html` y `login.js`
- Gestión y visualización de transacciones en `transactions.html` y `transactions.js`
- Depósito de fondos mediante el formulario en `deposit.html` y `deposit.js`
- Envío de dinero con confirmación y resumen en `sendmoney.html`
- Renderizado dinámico de lista de operaciones usando `transactions.fragment.js`
- Estilos responsivos y personalizados en `assets/css/style.css`
- Navegación entre páginas principales para experiencia de usuario fluida
